# TDD Analytics Domain - Process Flow

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TDD ANALYTICS DOMAIN PROCESS                      │
└─────────────────────────────────────────────────────────────────────┘

                         ┌──────────────┐
                         │    START     │
                         └──────┬───────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────────┐
│  PHASE 1: RED (Write Tests First)                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌─────────────────────┐    ┌─────────────────────┐                 │
│  │  Write Unit Tests   │    │ Write Integration   │                 │
│  │  (mocked responses) │    │ Tests (real API)    │                 │
│  └──────────┬──────────┘    └──────────┬──────────┘                 │
│             │                          │                             │
│             └──────────┬───────────────┘                             │
│                        ▼                                             │
│              ┌─────────────────────┐                                 │
│              │   Run Pytest        │                                 │
│              │   (expect FAIL)     │                                 │
│              └──────────┬──────────┘                                 │
│                         │                                            │
└─────────────────────────┼────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────────────┐
│  PHASE 2: GREEN (Implement)                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  Implement Analytics Class                                   │    │
│  │  ├── __init__(client)                                       │    │
│  │  ├── get_report_headers(report_path) → Dict                 │    │
│  │  └── fetch_report_rows(report_path, limit, max_rows) → Gen  │    │
│  └──────────────────────────────┬──────────────────────────────┘    │
│                                 │                                    │
│                                 ▼                                    │
│              ┌─────────────────────┐                                 │
│              │   Run Pytest        │                                 │
│              │   (expect PASS)     │                                 │
│              └──────────┬──────────┘                                 │
│                         │                                            │
└─────────────────────────┼────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────────────┐
│  PHASE 3: REFACTOR                                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐   │
│  │  Update Exports  │  │  Code Quality    │  │  Final Verify    │   │
│  │  (__init__.py)   │→ │  Check           │→ │  (smoke test)    │   │
│  └──────────────────┘  └──────────────────┘  └────────┬─────────┘   │
│                                                        │             │
└────────────────────────────────────────────────────────┼─────────────┘
                                                         │
                                                         ▼
                                               ┌──────────────┐
                                               │   COMPLETE   │
                                               └──────────────┘

```

## Task Dependencies

```
writeUnitTestsTask ─────────┐
                            ├──► runPytestTask (RED)
writeIntegrationTestsTask ──┘         │
                                      ▼
                          implementAnalyticsTask
                                      │
                                      ▼
                          runPytestTask (GREEN)
                                      │
                                      ▼
                          updateExportsTask
                                      │
                                      ▼
                          codeQualityCheckTask
                                      │
                                      ▼
                          finalVerificationTask
```
